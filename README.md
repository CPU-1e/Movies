# Abdul Movies

A beautiful, ad-free movie discovery and streaming app with clean UI.

## Features

- 🎬 Browse popular, top-rated, upcoming, and now-playing movies
- 🔍 Search movies instantly
- 🖼️ High-quality movie posters from TMDB
- 📺 Links to free, legal streaming sources (YouTube, Tubi, Pluto TV, Crackle)
- 📱 Fully responsive design
- 🌙 Modern dark theme
- ⚡ Fast and lightweight

## Free Streaming Sources

All streaming links point to **free, legal** platforms:
- **YouTube** - Free movies and trailers
- **Tubi** - Free ad-supported streaming
- **Pluto TV** - Free live TV and movies
- **Crackle** - Free movies and shows
- **IMDb** - Movie information
- **JustWatch** - Find where to watch

## Setup Instructions

### 1. Get a Free TMDB API Key

1. Go to [https://www.themoviedb.org/signup](https://www.themoviedb.org/signup)
2. Create a free account
3. Go to Settings > API and request an API key
4. Open `app.js` and replace `YOUR_TMDB_API_KEY` with your actual key

### 2. Deploy to GitHub Pages (Free Hosting)

1. Create a GitHub account at [https://github.com](https://github.com)
2. Create a new repository named `AbdulMovies`
3. Upload all files from this folder
4. Go to Settings > Pages
5. Select "main" branch and save
6. Your site will be live at `https://yourusername.github.io/AbdulMovies`

### 3. Custom Domain (Optional)

**Free subdomain options:**
- `.github.io` (included with GitHub Pages)
- `.netlify.app` (free with Netlify)
- `.vercel.app` (free with Vercel)

**To buy a custom domain:**
- `.net` domain: ~$10-15/year at Namecheap or GoDaddy
- `.org` domain: ~$10-15/year
